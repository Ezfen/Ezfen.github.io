BST and AVL

看到题目就知道，这篇文章要讲的就是算法中经常会出现的两种树。

计算机科学中，树是一种抽象数据类型，或是实现了类似抽象数据类型的数据结构。我们都知道，树是由有限个节点组成的具有层次关系的集合，每个节点都可能有零个或多个节点，但今天的主角 BST 和 AVL 却只有最多两个节点，却是算法中非常受欢迎的数据结构类型。

二叉树

二叉树是最常见的树了，

比如我们在 LeetCode 的时候经常可以看到二叉树的题目，比如这道题目 - 701. Insert into a Binary Search Tree， LeetCode 对二叉树的代码定义十分的简单，但简单的背后可以延伸的算法思维却是十分复杂的：

/* 
 * Swift
 * Definition for a binary tree node. 
 */
public class TreeNode {
    public var val: Int
    public var left: TreeNode?
    public var right: TreeNode?
    public init(_ val: Int) {
        self.val = val
        self.left = nil
        self.right = nil
    }
}
